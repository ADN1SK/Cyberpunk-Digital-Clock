# Digital Clock

A simple digital clock web application created as a personal project by ADN1SK.

## Overview

This project is a lightweight digital clock built with standard web technologies (HTML, CSS, and JavaScript). The interface displays the current time in a visually appealing manner, updating every second. It is designed as a learning exercise, showcasing a combination of frontend development skills and real-time DOM manipulation.

## Features


- **Live Time Display:** Shows current hours, minutes, and seconds, updating continuously.
- **Minimalist Design:** Clean and readable interface with custom styling using CSS.
- **Responsive Layout:** Works across devices due to simple, adaptable HTML/CSS structure.

## Getting Started

To run this project:

1. **Download or clone** the repository:
   ```bash
   git clone https://github.com/ADN1SK/Digital_clock.git
   ```
2. **Open `index.html`** in your web browser.

No additional dependencies or build steps are required.

## Project Structure

- [`index.html`](https://github.com/ADN1SK/Digital_clock/blob/main/index.html): Main webpage container for the clock.
- [`style.css`](https://github.com/ADN1SK/Digital_clock/blob/main/style.css): Custom styles for clock appearance.
- [`index.js`](https://github.com/ADN1SK/Digital_clock/blob/main/index.js): JavaScript logic to update and display the time.
